# Inspirational Quotes Discord Bot - Usage Instructions

## Introduction

The Inspirational Quotes Discord Bot is designed to share inspirational quotes in your Discord server. Follow the instructions below to set up and use the bot effectively.

## Setup Instructions

1. **Invite the Bot to Your Server**  
Use the bot's invite link to add it to your Discord server. Ensure you have the necessary permissions to manage bots.

2. **Grant Required Permissions**  
The bot requires the following permissions:
    - Send messages
    - Upload images
    - Embed links

3. **Configure the Bot**  
    - Use the `/setup` command to configure the bot for your server.
